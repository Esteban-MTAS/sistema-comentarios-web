# Sistema de comentarios web

Proyecto grupal desarrollado con HTML, CSS, JavaScript y PHP para recibir, almacenar y mostrar comentarios de usuarios.

## Tecnologías usadas
- HTML
- CSS
- JavaScript
- PHP
- Bootstrap

## Funcionalidades
- Formulario para publicar comentarios
- Almacenamiento de comentarios en servidor
- Visualización elegante de comentarios
- Publicación del proyecto en servidor remoto

## Estructura del proyecto
- `index.html`: estructura principal
- `css/style.css`: estilos
- `js/app.js`: validaciones e interacción
- `php/guardar_comentario.php`: recibe y guarda comentarios
- `php/mostrar_comentarios.php`: recupera y muestra comentarios
- `data/comentarios.json`: almacenamiento de datos
